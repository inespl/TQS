
Testes Unitários
- Devem ser fine grained, testar uma só coisa
- Devem ser rápidos
- Devem ser independentes entre si 
- Devem ser feitos/testados de forma isolada
- Não devem considerar outros testes (mas se for mesmo preciso, usar a anotação @Nasted para fazer testes estruturados)
